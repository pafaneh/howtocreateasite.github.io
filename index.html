Instructions for Creating a Site with GitHub

Below is a list of step by step instructions that you can follow to easily create a site and upload it to Github.

  • Step One:

    The first step in the cycle of creating a site and uploading it to GitHub is to create a directory withing your local system that will hold all the files such as html, css & js files.

    The file structure should look something like this:

    • Index.html
    • css/styles.css
    • js/javascript.js
    Now you have the basics structure of your starting site files.

  • Step Two:

    Using the combination of html, css and javascript if desired compile the page you want to make available for the world to see using an editor such as VS Code or similiar. Be sure to link your css & js files so that they are connected properly.

  • Step Three:

    Once you are fully happy with the outcome of your newly created page, and after you have tested it locally to make sure all links etc are working properly. It is now time to prepare to upload them to GitHub. The first thing you have to do is to log into your account.

  • Step Four:

    Once logged it it you be on your overview page. In order to upload your files you would need to go to the "Repositories" tab.

  • Step Five:

    Once inside your respository click on "new" to create a new respository. Next you would complete these steps:

    • Give it a Respsitory Name. Example: howtocreateasite.github.io
    • Next give it an optional description
    • Select rather you want it to be a public or private upload
    • Select Add a Readme file in the event you would like to include one.
    • Then click "Create Respository"
    • Next you would select the link text that says "uploading an existing file". This will open a drag and drop interface where you will upload all site files. Then select "Commit Changes"
    • Then your files will upload to your newly created Repo.

  • Step Six:

    The way to share the files so anyone can view it in their web browser, is you would provide a link to the named git repo such as: howtocreateasite.github.io.
    It can take up to 10 minutes for it to publish. So don't be concerned if you are unable to see it immediately.
